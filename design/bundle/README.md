# Beauty Empire Design Bundle

This bundle contains all the assets required to work with the Beauty Empire landing page design.

## Structure
- `pencil-new.pen`: The primary design file (usable with Pencil / Agentic Code tools).
- `BeautyEmpire_Final_Design.pdf`: The high-quality vector export of the design.
- `images/`: High-resolution AI-generated assets used in the design.
- `fonts/`: Lato font family (Regular, Bold, Italic, Light, Thin) required for consistent typography.

## How to use in Figma
1. **Install Fonts**: Install all files from the `fonts/` directory onto your computer.
2. **Import Design**: Use the `BeautyEmpire_Final_Design.pdf` file. If a direct drop into Figma fails, use the **"PDF to Figma"** plugin or convert it to **SVG** using an online tool.
3. **Images**: All used images are provided in the `images/` folder if you need to replace or re-scale them manually.
